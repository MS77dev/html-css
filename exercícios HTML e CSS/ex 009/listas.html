Criando listas

Listas ordenadas

Nestas listas, a ordem faz diferença

  1. Fazer feira
  2. Consertar armário
  3. Arrumar setup de café
  4. Lavar roupas
  5. Varrer casa
  6. Arrumar sofá e sala

É possível mudar a classificação com o comando Type como nos exemplos a seguir

  1. abacate
  2. banana
  3. aveia

  1. Barzinho
  2. Correr no parque
  3. Cozinhar

É possível ainda fazer listas aninhadas utilizando uma sublista dentro da lista original

  1. capítulo 1
  2. capítulo 2
  3. capítulo 3
    1. Histórias
    2. Eventos
    3. Testes
  4. capítulo 4

Criando listas não ordenadas

Nestas listas não há sequência ou numeração, apenas aqueles pontinhos ou tracinhos como nos exemplos a seguir:

Usando o comando type = square

  • leite
  • pão
  • ovos
  • carne

usando o comando type = "disc"

  • leite
  • pão
  • ovos
  • carne
